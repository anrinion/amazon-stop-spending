# Amazon Spending Brake

A Tampermonkey userscript designed to slow down impulse buying on Amazon by limiting access and encouraging the use of a persistent shopping list.

## Screenshots

![Screenshot of the intervention overlay](screenshots/overlay.png)

![Screenshot of the shopping list widget](screenshots/widget.png)

## Features

- **Weekly Limits**: Restricts access to a set number of sessions per week (configurable, defaults to 3).
- **Session Timer**: Active sessions last for 1 hour, allowing uninterrupted browsing across pages and tabs.
- **Shopping List**: Add items to a persistent list with checkboxes to track what you actually need.
- **Theming**: Support for light and dark modes.

## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension for your browser.
2. Install the script by paste the code into a new Tampermonkey script.
